# TerraRisk Platform

Professional monorepo for **TerraRisk AI** — intelligent terrain and geospatial risk analytics.

## Repository Structure

```
terrarisk-platform/
├── frontend/           # Next.js web application
├── backend/            # API services (coming soon)
├── docs/               # Documentation
├── data/               # Datasets and sample data
├── infrastructure/     # IaC and cloud configuration
├── docker/             # Docker configurations
├── scripts/            # Utility and automation scripts
├── .github/            # GitHub Actions and templates
└── README.md
```

## Quick Start

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Build

```bash
cd frontend
npm run build
```
